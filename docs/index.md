# Software defined Infrastructure

Project for the module "Software defined Infrastructure" at Hochschule der Medien Stuttgart.

## Goal

- Learn the principles and the core idea of software defined infrastructure
- Learn to use Terraform
- Setup VM to use as a webserver
- Learn to use the Hetzner Cloud
- Learn to use Pulumi
